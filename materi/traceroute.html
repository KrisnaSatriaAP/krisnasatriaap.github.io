6. Melacak Rute Perjalanan Paket Data (Traceroute)

Pernahkah Anda penasaran bagaimana data internet dari laptop Anda bisa sampai ke server besar seperti Google atau Facebook? Kita bisa melacak jalurnya menggunakan utilitas diagnosis jaringan bernama Traceroute.

Fungsi Utama Traceroute:

Traceroute memetakan satu per satu lompatan router (hops) yang dilewati oleh paket data Anda dari komputer rumah, melewati server penyedia internet (ISP), hingga sampai ke server tujuan akhir. Alat ini sangat berguna untuk melakukan troubleshooting guna mengetahui di titik server atau router mana koneksi internet Anda mulai mengalami kelambatan (latensi tinggi) atau putus total.

Cara Menjalankannya:

  1. Buka aplikasi Command Prompt (Windows) atau Terminal (Mac/Linux).
  2. Jika Anda menggunakan Windows, ketik perintah berikut: tracert google.com lalu tekan Enter.
  3. Jika Anda menggunakan Mac atau Linux, ketik perintah: traceroute google.com lalu tekan Enter.

Cara Membaca Hasil Cetak (*Output*) Traceroute:

  • Nomor Lompatan (Hop Index): Kolom angka berurutan di sebelah kiri (1, 2, 3, dst.) menunjukkan urutan jumlah router perantara yang dilalui paket data.
  • Waktu Respons (RTT): Tiga kolom waktu dalam satuan milidetik (ms) menunjukkan kecepatan respon bolak-balik dari router tersebut. Semakin kecil angkanya, jaringan semakin bagus.
  • Domain/IP Address: Kolom paling kanan menunjukkan nama host nama domain atau alamat IP dari server/router perantara yang sedang dilewati pada lompatan tersebut.
  • Tanda Bintang (*): Jika muncul tanda bintang dengan pesan "Request timed out" pada lompatan tertentu, berarti router perantara tersebut sedang sibuk, mengalami gangguan, atau sengaja memblokir protokol pelacakan demi keamanan.
