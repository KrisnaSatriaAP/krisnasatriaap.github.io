1. Mengenal IP Address secara Mendalam

Internet Protocol Address (IP Address) adalah identitas angka unik yang dimiliki oleh setiap perangkat (komputer, HP, router) agar bisa saling berkomunikasi di dalam jaringan lokal maupun internet. Anggap saja IP address seperti alamat rumah untuk pengiriman paket data.

Fungsi Utama IP Address:

  • Alamat Identifikasi Alat (Host): Mengetahui identitas spesifik suatu perangkat di jaringan.
  • Alamat Lokasi Jaringan: Menunjukkan rute atau jalur mana yang harus dilewati data agar sampai ke perangkat tujuan.

Perbedaan IPv4 vs IPv6:

  • IPv4: Terdiri dari 32-bit angka desimal yang dipisahkan titik. Contoh: 192.168.1.1. Kapasitasnya terbatas sekitar 4,3 miliar alamat di seluruh dunia.
  • IPv6: Generasi baru dengan 128-bit kombinasi angka dan huruf (heksadesimal) dipisahkan titik dua. Contoh: 2001:db8::ff00:42:8329. Dibuat karena stok alamat IPv4 dunia sudah mulai habis.

Cara Cek IP Address Sendiri (Windows):

  1. Tekan tombol kombinasi Windows + R di keyboard Anda, ketik cmd, lalu tekan Enter.
  2. Ketik perintah ipconfig di dalam Command Prompt, lalu tekan Enter.
  3. Cari adapter jaringan aktif yang sedang Anda gunakan (Wi-Fi atau Ethernet), lalu lihat pada baris IPv4 Address untuk mengetahui alamat IP komputer Anda.
