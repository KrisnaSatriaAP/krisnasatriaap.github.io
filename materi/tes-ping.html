2. Panduan Lengkap Tes Koneksi (Ping)

PING (Packet Internet Groper) adalah perintah utilitas yang digunakan untuk mengecek integritas atau kualitas koneksi antara perangkat Anda dengan server target di internet menggunakan protokol ICMP.

Cara Melakukan Ping:

Buka Command Prompt (Windows), lalu ketik: ping google.com dan tekan Enter.
