2. Panduan Lengkap Tes Koneksi (Ping)

PING (Packet Internet Groper) adalah perintah utilitas yang digunakan untuk mengecek integritas atau kualitas koneksi antara perangkat Anda dengan server target di internet menggunakan protokol ICMP (Internet Control Message Protocol).

Cara Melakukan Ping:

  1. Buka aplikasi Command Prompt (Windows) atau Terminal (Mac/Linux) di komputer Anda.
  2. Ketik perintah berikut: ping google.com atau ping 8.8.8.8 lalu tekan Enter.
  3. Sistem secara default akan mengirimkan 4 paket data uji coba dan menunggu balasan dari server tujuan.

Cara Membaca Hasil Respon Ping:

  • Reply from ... bytes=32 time=15ms TTL=56 : Koneksi jaringan berjalan normal dan sukses. Nilai angka pada time menunjukkan delay/latensi pengiriman data dalam satuan milidetik (ms). Semakin kecil nilainya, semakin cepat koneksi Anda.
  • Request Timed Out (RTO) : Paket data uji coba hilang di tengah jalan atau tidak mendapatkan balasan. Hal ini menandakan adanya gangguan kabel fisik, sinyal buruk, atau server target yang dituju sedang sibuk/memblokir sistem keamanan ping.
  • Destination Host Unreachable : Komputer lokal Anda sama sekali tidak dapat menemukan jalur rute menuju alamat IP target. Biasanya terjadi akibat kesalahan konfigurasi kabel LAN lokal, masalah pada router, atau Wi-Fi laptop yang terputus.
