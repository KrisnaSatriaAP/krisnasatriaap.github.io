6. Melacak Rute Perjalanan Paket Data (Traceroute)

Pernahkah Anda penasaran bagaimana data internet dari laptop Anda bisa sampai ke server besar seperti Google atau Facebook? Kita bisa melacak jalurnya menggunakan utilitas bernama Traceroute.

Fungsi Utama Traceroute:

Traceroute memetakan satu per satu lompatan router (*hops*) yang dilewati oleh paket data Anda dari rumah hingga sampai ke server tujuan. Alat ini sangat berguna untuk mengetahui di server atau titik router mana koneksi internet Anda mulai mengalami kelambatan atau putus.

Cara Menjalankannya:

Buka Command Prompt di Windows, lalu ketik perintah tracert google.com dan tekan Enter. Layar akan menampilkan daftar nomor IP router perantara beserta waktu responsnya dalam satuan milidetik (ms).
