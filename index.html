Belajar Jaringan Komputer Jadi Mudah

Kumpulan panduan praktis dan tutorial dasar mengenai dunia jaringan internet dan infrastruktur komputer.

Daftar Tutorial Dasar

1. Mengenal IP Address

IP Address adalah alamat identitas unik setiap perangkat di dalam jaringan internet.

  1. Buka Command Prompt (cmd) di Windows.
  2. Ketik perintah ipconfig lalu tekan Enter.
  3. Cari baris IPv4 Address untuk melihat IP komputer Anda.

2. Cara Tes Koneksi (Ping)

Perintah Ping digunakan untuk memeriksa apakah komputer Anda terhubung ke server target.

  1. Buka terminal atau Command Prompt.
  2. Ketik ping google.com lalu tekan Enter.
  3. Jika muncul balasan Reply from..., koneksi Anda normal.

3. Urutan Kabel UTP (Crimping)

Panduan menyusun kabel LAN dengan standar internasional T568B (Straight-through).

  1. Urutan warna: Putih-Orange, Orange, Putih-Hijau, Biru.
  2. Lanjutan: Putih-Biru, Hijau, Putih-Cokelat, Cokelat.
  3. Ratakan ujung kabel, masukkan ke konektor RJ45, lalu jepit.
